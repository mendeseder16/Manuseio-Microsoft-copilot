# Manuseio-Microsoft-copilot

Início e Configuração Inicial
Acesse o Portal: Entre em copilotstudio.microsoft.com com sua conta Microsoft 365.
Crie um Novo Copiloto: Na aba Criar, selecione Novo Copiloto. Você pode descrever o que deseja que o assistente faça em linguagem simples para que a plataforma gere uma configuração inicial.
Identidade: Defina o nome, o idioma (ex: Português) e adicione um ícone para personalizar a marca do seu assistente.

2. Configuração de Fluxos de Conversa (Tópicos)
Os tópicos são árvores de diálogo estruturadas que guiam a interação com o usuário.
Criar Tópicos: Vá em Tópicos e crie novos do zero ou use o Copilot integrado para gerar um tópico descrevendo o objetivo.
Gatilhos (Triggers): Defina frases ou eventos que iniciam a conversa (ex: "Como posso recuperar minha senha?").
Nós de Conversa: No designer visual, adicione nós como Mensagem (enviar texto), Pergunta (obter dados do usuário) e Condição (direcionar o fluxo com base em respostas).

3. Integração com Serviços Externos (Ações e Conhecimento)
Para tornar o assistente funcional e capaz de automatizar tarefas, conecte-o a dados e sistemas.
Fontes de Conhecimento: Adicione sites públicos, arquivos (PDF, Word), listas do SharePoint ou bancos de dados do Dataverse para que a IA gere respostas fundamentadas nesses dados.
Ações e Fluxos: Utilize conectores e fluxos do Microsoft Power Automate para realizar tarefas, como consultar um CRM, abrir chamados de TI ou enviar e-mails via Outlook.
APIs: Configure conectores personalizados para buscar dados em tempo real de APIs externas.

4. Personalização e Respostas Inteligentes
Respostas Generativas: Habilite o "Conversational Boosting" para que o assistente responda de forma natural mesmo quando não houver um tópico específico configurado, usando as fontes de conhecimento.
Instruções de Persona: Defina o tom de voz (ex: amigável, formal), o comprimento das respostas e restrições sobre o que ele não deve discutir.

5. Teste e Publicação
Painel de Teste: Use o painel à direita para interagir com o copiloto em tempo real e verificar se os fluxos e integrações funcionam como o esperado.
Publicação: Quando estiver pronto, publique o assistente em diversos canais, como Microsoft Teams, websites corporativos ou no chat do Microsoft 365
